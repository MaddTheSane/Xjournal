LJEntry

Abstract: Represents a LiveJournal entry.

The LJEntry class represents a LiveJournal entry. An entry is either unassociated, associated with a journal, or posted to a journal. When entries are created using init they are unassociated. You can associate them with a journal by calling the setJournal: method. The entry is finally posted by calling saveToJournal. Entries that are obtained from LJJournal's getEntries... methods are posted entries.

unassociated journal == nil
associated journal != nil, itemID == 0
posted journal != nil, itemID != 0
