Using the Glossary

Using Xjournal's Glossary panel, you can easily insert text clippings into your posts. To open the Glossary, select the Glossary option from the Window menu. The Glossary is populated by text files located in the Glossary folder. To open this folder in the Finder, click the Open Folder button.

You can add a new text clipping by saving a plain text file in the Glossary folder. For example, you could use TextEdit to create your clipping and save it. Then, click the Refresh button in Xjournal's Glossary panel. The new clipping will be listed by its filename. Select the clipping and then click the Insert button to paste the text in your post.
